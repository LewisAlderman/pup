# Pup (@lewald/pup)

Pick appropriate Github PR templates on the fly

## Install

```bash
$ npm install --global @lewald/pup
```

## CLI

```
$ pup --help

	Usage
	  $ pup

	Options
		--confirm  Should program get confirmation before running
		-c  Should program get confirmation before running

	Examples
	  $ pup -c
```
